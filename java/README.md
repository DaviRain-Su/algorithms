# 算法

## chapter1: basic

算法的定义：一种有限、确定、有效的并适合用计算机程序来实现的解决问题的方法。

算法是计算机科学的基础，是这个领域的核心。算法是一种解决问题的方法，是一种思维活动，是一种设计和分析的技术。

欧几里得算法

自然语言描述： 计算两个非负整数p和q的最大公约数：若q是0，则最大公约数为p。否则，将p除以q得到余数r，p和q的最大公约数即为q和r的最大公约数。

```java
public static int gcd(int p, int q) {
    if (q == 0) return p;
    int r = p % q;
    return gcd(q, r);
}
```
